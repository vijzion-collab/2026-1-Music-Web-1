# 2026-1 Music Web 1

Elite Sync Licensing for Visual Media. 100% Stem-Enabled. Broadcast Ready. Experience the future of production music cataloging.

## Project Overview
This project is a high-performance music catalog platform designed for high-stakes sync licensing (TV, Film, Commercials). Modeled after industry-leading catalogs like BrandXMusic, it features a modular architecture designed to provide music supervisors with a seamless search and licensing experience.

## Key Features
- **Full Stem Accessibility:** Every track includes a dropdown for alt versions and stems.
- **Broadcast-Ready Metadata:** Integrated BPM, Key, Mood, and Genre tags for rapid search.
- **Waveform-Driven Player:** Real-time visual audio playback.
- **Cinematic UI/UX:** A premium, dark-themed interface built with React and Vite.

## Tech Stack
- **Frontend:** React.js / Vite
- **Styling:** Vanilla CSS (Modular Design System)
- **Audio:** WaveSurfer.js
- **Icons:** Lucide React
